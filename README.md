# Short-Term-Solar-Power-Forecasting
This project focuses on short-term forecasting of solar photovoltaic (PV) power generation using a hybrid CNN–GRU deep learning model. The model is trained on real-world data from a solar plant in Germany, using features such as solar irradiance, temperature, and cloud coverage to improve forecasting accuracy for smart grid applications.
A hybrid deep learning architecture is implemented to effectively learn complex relationships within the data. Convolutional Neural Networks (CNNs) are used for feature extraction, while Gated Recuurent Units (GRU) networks model temporal dependencies in PV power output. The performance of the proposed model is evaluated using standard error metrics such as RMSE and MAE and is compared against conventional forecasting approaches.
The results demonstrate improved forecasting accuracy, highlighting the potential of hybrid deep learning models for short-term solar PV power prediction and their application in smart grid and renewable energy systems.
